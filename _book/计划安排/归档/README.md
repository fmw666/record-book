
#  归档

### 归档规则

1. 每日晚上写第二天计划，每周周日晚上额外整理当周归档

1. 每周归档结构命名规范：

    ```md
    .
    ├── 0821-0827
    │   ├── 0821.md
    │   ├── ...
    │   └── 0827.md
    ├── 0828-0903
    ...
    ```

    如果跨年，需要进行年归档

    ```md
    .
    ├── 2022
    │   ├── 1221-1227
    │   │   └── ...
    ├── 2023
    ...
    ```
