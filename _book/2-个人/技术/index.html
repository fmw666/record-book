Python 面试

Python 服务端工程师每天做什么？

  • 实现网站后台业务逻辑
  • 为网站/客户端（Android, iOS）提供接口
  • 为产品/运营等提供后台网站工具，比如后台管理系统

Python 服务端工程师有什么特点？

  • 后端知识又多又杂。编程语言/数据库/算法/网络/架构...
  • 工作多样。写接口/写网站/写工具/写前端/写脚本/运维/测试/部署...

体系：
