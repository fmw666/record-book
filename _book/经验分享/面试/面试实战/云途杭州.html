整体概述

面试 Python 杭州

公司全称：云途（杭州）智能科技有限公司

注册资本：100 万人民币

成立时间：2019-11-14

公司地址：杭州西湖区杭州华星科技大厦（东南门）9 楼

团队规模：10 人左右开发团队

业务范围：智慧城市系统与数字交通服务提供商。

工作时间：9:00-20:30，其中 中午 12:00-13:30 以及 15:00-15:15 休息，大小周

晋升方式：通过参与立项项目获取“经验值”，累计一定的“经验值”可以申请升职。

五险一金：杭州标准的最低系数缴纳。

薪资结构：基本工资（包含绩效工资）+ 立项项目奖金 + 年终奖

最终定薪：14.5k 14薪

面试时间

一面 2023-08-16 19:00 腾讯会议 周经理

二面 2023-08-19 17:00 腾讯会议 储总

个人评价

无额外餐补类的福利补贴，也无加班费，工作时间实在太长。

虽然所处地段很好，位于杭州较好的位置，而且做得内容是有一定前景的。

面试题目

一面

  1. 你工作的主要内容是什么？

  2. 你要如何实现一个较难的新需求，你的思路是什么？

    1. 先进行需求拆分，罗列出涉及到哪些业务模块？
    2. 根据拆分的结果，先咨询上级或同事，是否有类似需求或项目已经实现过的，如果有的话，多学习和了解之前的实现方案，然后再根据自己的理解和思考，结合之前的实现方案，来实现自己的需求。
    3. 如果没有的话，就自己根据每块业务进行技术可行性评估，以及实现所需时长评估。
    4. 同上级确认该评估结果的合理性，然后再根据评估结果，来决定是否实现该需求，或者如何安排工作计划。比如是一个人多少天完成，还是几个人共同处理。
    5. 如果是多人共同处理，那么就需要进行任务拆分，然后再根据每个人的技术能力和工作量，来安排每个人的任务。
    6. 根据需求选择合适的技术方案，然后再根据技术方案，来进行技术实现。

  3. 你为什么离职？

  4. 数学建模奖项？主要参与了哪些部分？你认为占比多少？
  5. 讲讲你项目中最难的一点
  6. 为什么使用 Protobuff 而不是 json？
  7. fastapi 的异步是如何实现的？
  8. 如果要实现一个表单提交的 post 请求，如何用 fastapi 去做？
  9. 基于装饰器去实现 fastapi 路由时，如果要为该路由添加日志，使用日志装饰器，该在路由装饰器上面还是下面
  10. 前后端项目中认证方式？
  11. token 的生成原理？
  12. 如何部署应用？
  13. docker 如何部署和发布容器？
  14. 如何实现跨域？
  15. 目前的一个职业规划是啥？

  2. 你最近一年做了什么？
